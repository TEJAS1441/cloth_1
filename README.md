
# Avatar BLE Web App

## Overview
This program streams IMU data via Bluetooth and visualizes a 3D avatar's movement in a web application.

## Running the Program

1. Ensure your Bluetooth of the device is on and ready to be paired. 
2. Activate virtual environment Run ble_client.py file --> python3 python/ble_client.py 
3. Run app : python3 python/app.py
4. Open your browser and navigate to:
    ```
    http://localhost:8000/index.html

## Features
- Real-time 3D avatar animation
- IMU data reception via Bluetooth
- Web-based visualization on port 8000

## Requirements
- Python 3.12
- Bluetooth-capable device
- Modern web browser
